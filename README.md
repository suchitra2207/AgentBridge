# 🚀 AgentBridge — Multi-Agent AI System

## 📌 Overview
AgentBridge is a multi-agent AI system designed to support employees and generate workplace insights using AI.

---

## 💡 Problem
Workplace issues like stress, burnout, and communication gaps are hard to detect early using traditional HR systems.

---

## 🤖 Solution
AgentBridge uses multiple AI agents:

- Employee Support Agent → provides guidance  
- Workplace Analysis Agent → classifies issues  
- Resource Agent → retrieves policies  
- Insights Agent → identifies patterns
